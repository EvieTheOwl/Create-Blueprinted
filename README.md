# Create: Blueprinted

Originally made for the [Brassworks SMP](https://brassworks.opnsoc.org/), this mod is free and open for anyone to use!

Create: Blueprinted allows you to render your Create mod schematics into high-resolution, isometric PNG images straight from the game.
___
## Features
* **Isometric Renders:** Easily render your `.nbt` schematics to high-resolution isometric PNGs directly from the Schematic Table UI.
* **UI Bug Fix:** Fixes a bug in the Create Schematic Table screen where long schematic names can overflow out of the textbox bounds.
___
## Usage
### In-Game UI
You can render schematics directly from the Schematic Table. Just select your schematic and click the new render button!

### Commands
If you need more control over the output (like changing the render orientation or the resolution quality), you can use the built-in command:
```bash
/renderschem <filename> [orientation] [quality]
```
